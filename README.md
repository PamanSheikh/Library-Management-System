# 📚 Library Management System (FastAPI + PostgreSQL)

A simple backend-based Library Management System built using **FastAPI, PostgreSQL, Pydantic, and psycopg2**.  
This project performs full CRUD operations on books using REST APIs.

---

## 🚀 Features

- Add a new book
- View all books
- View a single book by ID
- Update book details (PUT)
- Partially update book (PATCH)
- Delete a book
- PostgreSQL database integration
- FastAPI Swagger UI testing

---

## 🛠️ Tech Stack

- FastAPI
- Uvicorn
- Pydantic
- psycopg2
- PostgreSQL
